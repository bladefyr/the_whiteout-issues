# Pagophobia — Issue Tracker

This is the official bug tracker and feature request repository for **Pagophobia**, a Minecraft mod. 

To protect the mod's source code, this repository is used **strictly for tracking issues**. No code is hosted here.

---

## 📥 Download the Mod
Make sure you are playing on the latest version before reporting a bug!
* [Download on Modrinth](https://modrinth.com/mod/pagophobia)
* There's no plans to release this mod on Curseforge at all. If you see this mod on there, please report it and DO NOT DOWNLOAD IT! Thank you.

---

## 🪲 How to Report a Bug
Please follow these steps before opening a new issue:

1. **Search Existing Issues:** Check the [Issues tab](../../issues) to see if someone else has already reported your problem.
2. **Use the Template:** Click "New Issue" and fill out the provided form completely.
3. **Provide Logs:** **I cannot fix crashes without logs.** Please upload your `crash-report` or `latest.log` to [Pastebin](https://pastebin.com) or [Mclo.gs](https://mclo.gs) and paste the link in your report.

Thank you for helping improve Pagophobia!
